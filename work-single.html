Description

Application web développé par Laravel de e-commerce pour la gestion de vente de produit. Le client peut lister les produits disponible (Rechercher, filtrer) , remplir le panier, faire une commande, suivre sa commande. L'administrateur peut gérer les utilisateurs, gestion des produits, gestions des catégories, gestion des commandes, modifier son profil.
